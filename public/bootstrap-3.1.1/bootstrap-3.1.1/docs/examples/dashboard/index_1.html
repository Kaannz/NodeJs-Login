Trình bày các cách để tích hợp Bootstrap vào một trang HTML/CSS bình thường trong điều kiện đang có kết nối Internet? Trong Bootstrap đã được biên dịch gồm có những gì?

Có 2 cách để sử dụng bootstrap: Đó là down về và để chung thư mục với file html. Cách khác là ta có thể sử dụng đường dẫn như sau: ( với điều kiện là kết nối internet)

<!-- Kết nối với file bootstrap css gần nhất -->
<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css">

<!-- Kết nối với file bootstrap jquery gần nhất -->
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script>

<!-- Kết nối với file bootstrap javascript gần nhất -->
<script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/js/bootstrap.min.js"></script>

Trong bootstrap biên dịch gồm có 3 file css, javascript và jquery
